Η εφαρμογή δημιουργεί ένα κέλυφος (shell) το οποίο μπορεί να λειτουργήσει με δύο τρόπους.
Σε inreractive mode και σε Batch mode.

Σε interactive mode ($ ./argiShell) εμφανίζεται το prompt Argiropoulou_8491> και δέχεται μια ή περισσότερες εντολές διαχωρισμένες με τους χαρακτήρες ; ή &&.
Η έξοδος απο το shell γίνεται με την εντολή quit.

Σε batch mode δέχεται ένα αρχείο δέσμης (κειμένου) με τις εντολές που πρόκειται να εκτελεστούν.
($ ./argiShell batchfile )

To batch file μπορεί να περιλαμβάνει μία ή περισσότερες εντολές ανά γραμμή διαχωρισμένες με τους χαρακτήρες ; ή &&.
Το μέγιστο όριο όλων των χαρακτήρων της γραμμής είναι 512.
